# 业余无线电操作技术能力验证题库 转换工具

将 **业余无线电台操作技术能力验证题库（2025年版）** PDF 转换为结构化 TXT 和 CSV 格式的工具集。

## 项目结构

```
HAM/
├── pdf2txt.py          # PDF → TXT 转换（去除页码）
├── txt2csv.py          # TXT → CSV 解析转换
├── 业余无线电台操作技术能力验证题库（2025年版）/
│   ├── 总题库.pdf
│   ├── A类题库.pdf
│   ├── B类题库.pdf
│   ├── C类题库.pdf
│   └── 总题库附图标记.pdf
├── txt_output/         # 转换后的 TXT 文件（自动生成）
├── csv_output/         # 转换后的 CSV 文件（自动生成）
└── README.md
```

## 环境要求

- Python 3.11+
- [pypdf](https://pypi.org/project/pypdf/) （PDF 文本提取）

## 安装

```bash
# 创建虚拟环境（推荐）
python -m venv venv
venv\bin\python -m pip install pypdf

# 或直接安装
pip install pypdf
```

## 使用方法

### 1. PDF → TXT

递归扫描当前目录及子目录下的所有 PDF 文件，转换为 TXT 并输到独立目录（自动去除页码）。

```bash
venv\bin\python pdf2txt.py                    # 默认输出到 txt_output/
venv\bin\python pdf2txt.py --output my_txt    # 指定输出目录
```

输出示例：
```
处理 业余无线电台操作技术能力验证题库（2025年版）\总题库.pdf，输出 13192 行 txt，忽略 332 页码
```

### 2. TXT → CSV

交互式选择要转换的 TXT 文件，解析题库格式并输出 CSV。

```bash
venv\bin\python txt2csv.py                    # 默认输出到 csv_output/
venv\bin\python txt2csv.py --output my_csv    # 指定输出目录
```

交互示例：
```
找到以下TXT文件，请选择要转换的：

  [1] txt_output\业余无线电台操作技术能力验证题库（2025年版）\总题库.txt  (486.0 KB)
```

## 题库格式说明

TXT 文件使用带标记的文本格式，每条题目包含以下字段：

| 标记 | 字段     | 说明               |
|------|----------|-------------------|
| [J]  | 编号     | 题目编号（如 LY0001） |
| [P]  | 章节     | 章节编号（如 1.1.1）  |
| [I]  | 类型     | 题型/难度代码         |
| [Q]  | 题干     | 题目内容             |
| [T]  | 答案     | 正确答案选项          |
| [A]~[D] | 选项   | 四个选项             |
| [F]  | 图片     | 图片引用（如有）       |
